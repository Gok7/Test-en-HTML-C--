PAGE DE TEST D'UNE REQUETE UTILISANT LA METHODE GET

Avec cette méthode, le serveur récupère les données de la requête dans une variable QUERY_STRING que l'on peut lire avec la fonction C getenv.

Dans l'exemple suivant, la requête est envoyé au serveur en déclenchant l'exécution de l'exécutable req_get.cgi qui génére une page HTML contenant la chaîne reçue dans QUERY_STRING.
